Primeira pergunta

Instalação do Git no Windows

Para começar a instalação do Git no seu computador com sistema operacional Windows, abra o seu navegador e digite o seguinte endereço: git-scm.com/downloads. Uma vez na página de downloads do Git, clique no ícone do Windows, como na imagem.

Na página de opções de download, clique na opção de acordo com o tipo de computador que você usa.

Após clicar na opção, espere o download ser concluído corretamente e então, abra-o. Uma tela vai ser mostrada para indicar o destino para o diretório Git. O recomendado é que siga o estabelecido pelo sistema.

Selecione os componentes adicionais que você pretende instalar. Os que não forem selecionados não serão instalados. Quando pronto, clique em “Next”. Se estiver de acordo com o que for sendo estabelecido, continue clicando em “Next” durante a próxima etapa, mas se quiser alterar algo, o faça.

Aqui, há uma barra de seleção para os editores-base que o Git pode usar, já usando o Vim como o principal, embora não seja o melhor. Se precisar, pode mudar para outro editor, escolhendo uma das opções abaixo.

Na etapa seguinte, escolha se quer manter o branch inicial como “master” ou quer dar um nome personalizado para novos repositórios. Feito isso, siga clicando em “Next” por quatro etapas.

Escolha onde você quer usar o Git. Pelo Git Bash apenas, direto na linha de comando ou Git com ferramentas Unix do Prompt de Comando do computador. Siga avançando nas etapas até a última.

Na última tela, se estiver de acordo com as opções, clique em "Install" para concluir a instalação do Git.

Segunda pergunta

Instalação do Git no Linux

Começo do tutorial

No sistema Linux, abra o seu navegador e acesse o mesmo endereço de downloads: git-scm.com/downloads. Uma vez lá, dessa vez, clique no ícone de Linux/Unix.

Na página do Linux, copie o código indicado para a versão Linux que você está usando e a cole no terminal Linux. Em seguida, dê "Enter" e o processo de instalação vai começar.

Terminado o processo, execute um git version no terminal para verificar se realmente o Git foi instalado. Caso apareça a versão do Git, a operação foi feita com sucesso.
